# Desafío: HTML_Formularios

## Instrucciones

Crear un nuevo archivo dentro de la carpeta `03-Html_Formularios` llamado `desafio.html` donde deberás realizar lo siguiente:

```
I. Cree una página HTML básica
  A. Cree el diseño básico para una página HTML usando html, head y body.
  B. Agregue un elemento de título y asigne a su página el título de "Tarea HTML_Formularios de [Su nombre]".
  C. En el cuerpo crea.
    1. un elemento form que contiene 
      * Radiobuttons
        - input: radio con el name «idioma» y value «idiomaingles» y el texto «Inglés»
        - input: radio con el name «idioma» y value «idiomaaleman» y el texto «Alemán»
        - input: radio con el name «idioma» y value «idiomafrances» y el texto «Francés»
      * lista desplegable con el name «conector»
        - opción «USB» con el value usb          
        - opción «Paralelo» con el value paralelo
        - opción «PS2» con el value ps2
      * cuadros de texto
        - cuadro de texto con el texto «Nombre» y el name nombre
        - cuadro de texto con el texto «Apellidos» y el name apellidos
      * lista desplegable múltiple con el name «sexo»          
        - opción «Hombre» con el value hombre
        - opción «Mujer» con el value mujer.
      * checkboxes
         - checkbox con el name «sexo» , value «sexomujer» y el texto «Mujer»
        - checkbox con el name «sexo» , value «sexohombre» y el texto «Hombre».
      * control para indicar la fecha.
```
